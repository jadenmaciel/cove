## Key Research Findings

- `teardownAllPanels()` in Workspace.swift kills all terminal processes (line 10519)

- `TabManager.closeWorkspace()` already calls `workspace.teardownAllPanels()` (line 329)

## Architecture

Inline Rename:
```
Sidebar Row Double-Click
        |
        v
Set isEditing = true for that workspace
        |
        v
Show TextField instead of Text
        |
        v
onSubmit: update record.name + WorkspaceRegistryStore.save()
        |
        v
Sidebar re-renders with new name
```

Delete (Archive + Kill):
```
User clicks Delete / presses Backspace on selected workspace
        |
        v
If live workspace exists: workspace.teardownAllPanels() (D-11)
        |
        v
Call archive() on record (soft delete, not hard delete)
        |
        v
WorkspaceRegistryStore.save()
        |
        v
Sidebar re-renders without deleted workspace
```

    4. Ensure cancel (pressing Escape) reverts to original name and sets `isEditing = false`

    3. Confirm with user before delete (standard macOS dialog):
       - Title: "Delete Workspace?"
       - Message: "All terminals in this workspace will be closed."
       - Buttons: "Cancel", "Delete" (destructive)

    Note: Archive preserves undo capability (soft delete). Hard delete can be added later.
